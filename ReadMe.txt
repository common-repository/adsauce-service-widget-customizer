Accept self-serve Advertising, Sponsorships and Donations. Customize ad spots, business directory or social message board all in one place.

== Description ==

Perfect for schools, churches, non-Profits, local government, political campaigns, local business districts and Chambers of Commerce, bloggers, events, youth sports and hobbyists.  AdSauce is easy to use and does the lions share of the work. Use the AdSauce wizard at AdSauce.co to create your choice of platforms, then use the WordPress widgets and short code to place them where you want.  Optionally you can cut and paste the code generated by the wizard.

Features:

* Banner Ads
* Business Directory
* Social Message board
* All customizable platforms managed in one location
* Advertisers only enter their info once
* Self-Serve to your advertisers, sponsors or donors
* Notifies you when ads or messages are pending
* You retain 100% content control (for sensitive websites)
* Notifies advertisers when ads are expiring
* Reports metrics to you and to your advertisers
* Takes care of all accounting
* Sends your revenue to your pay-pal account
* Free to set up and no monthly fees
* No traffic minimums
* Use widget, short code or copy and paste code.
* Cloud Based
* No additional software needed
* No hardware upgrade needed
* Always using the latest version

== Installation ==

For installing AdSauce Service Plugin please follow the instruction:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->AdSauce to configure the plugin
